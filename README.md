# offline-pdf-tools
Process PDFs and images entirely in your browser. Merge PDFs, convert images to PDF, and combine files without sending data to external servers.
# Offline PDF Tools

🇺🇸 **English**

A lightweight, privacy-first PDF toolkit that runs entirely in your browser.

This project was created to provide a simple and secure alternative to online PDF tools, ensuring that your files never leave your computer.

## Features

- 📄 Merge multiple PDF files
- 🖼️ Convert images to PDF
- 📑 Combine PDFs and images into a single PDF
- 🔒 100% local processing (no file uploads)
- 🌐 Runs directly in your browser

## Privacy

All processing happens locally in your browser.

No files are uploaded, stored, or sent to external servers.
